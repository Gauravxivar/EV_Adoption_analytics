# Global EV Adoption Behavior Analytics Dashboard

End-to-end Data Analytics Project using MySQL, Python, and Power BI

### 📊 Project Overview

Analyzed a 50,000-row EV adoption dataset to identify customer adoption patterns, vehicle preferences, and demographic trends.

### 🛠 Tools Used

* MySQL – Data querying and business analysis

* Python (Pandas, Matplotlib) – Exploratory Data Analysis

* Power BI – Interactive dashboard development

* GitHub – Project version control and portfolio hosting

### 📁 Project Files

| File              | Description             |
| ----------------- | ----------------------- |
| ev_analysis.sql   | 18 business SQL queries |
| ev_eda.ipynb      | Python EDA notebook     |
| ev_dashboard.pbix | Power BI dashboard      |
| dashboard.png     | Dashboard screenshot    |

### 🔍 Key Insights

* 59% of respondents showed High EV Adoption Likelihood.

* Urban residents were significantly more likely to adopt EVs than rural residents.

* Higher income groups demonstrated stronger EV purchase intent.

* SUV and Sedan owners represented the largest potential EV customer segment.

### 📌 Dashboard Features

* KPI cards for key metrics

* Adoption likelihood analysis

* Vehicle type distribution

* City type comparison

* Income and education segmentation

* Interactive slicers and filters
